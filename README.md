# Planet Earth MC - 양조 마스터 헬퍼 (Brewing Master Helper)

이 프로그램은 마인크래프트 Planet Earth 서버의 양조 시스템을 위한 레시피 도감 및 공정 타이머입니다. Nuitka로 컴파일되었습니다.

## 1. 실행 방법 (How to Run)
1.  프로그램 폴더 내의 배치 파일(예: `main.exe`)을 실행하세요.

---

## 3. 주요 기능

### 상세 레시피 도감 (좌측 섹션)
* 드롭다운 메뉴에서 술 종류 선택 시 재료, 발효 시간, 증류 횟수, 숙성 조건 출력.
* 판매 가격 및 알코올 도수 정보 제공.

### 공정 타이머 (우측 섹션)
* **발효 시작:** 선택한 술의 표준 발효 시간(분 단위)으로 타이머 작동.
* **숙성 시작:** 게임 내 시간 시스템에 맞춰 20분당 1년으로 계산하여 타이머 설정.
* **상태 표시:** 진행 상황을 프로그레스 바와 실시간 카운트다운으로 확인 가능.

---

## 4. 트러블슈팅

* **실행이 되지 않는 경우:** 1. Python 3.10이 설치되어 있는지 확인하십시오.
    2. 배치 파일 내의 `SET PYTHONHOME` 경로가 실제 파이썬 설치 경로와 일치하는지 확인하십시오.
* **DLL 오류 발생 시:** PyQt6 패키지가 파이썬 `site-packages` 폴더 내에 정상적으로 존재하는지 확인하십시오.
---

## 5. 면책 사항 및 안내
이 프로그램은 Planet Earth MC 서버의 양조 데이터를 기반으로 제작되었습니다. 서버 업데이트에 따라 레시피 수치가 변동될 수 있습니다.
